Which extensions did you install? Why?

  - I chose all four extensions, since after I briefly looked into them, they seem to have a specific purpose that they specialize in one aspect that did not overlap with the another.

What was the most useful thing you learned?

  - REACT DEVELOPER TOOLS: While I am new to REACT, this should inspect props and states of components in the components menu.

  - REDUX DEVTOOLS: Interestingly, it can do "Time-Travel Debugging", which replays actions and can see how state changes per step.

  - JSON VIEWER: Can expand and collapse nested objects. Makes APIs responses readable.

  - LIGHTHOUSE: Runs a Performance Audit and shows a Performance score plus a Core Web Vitals. Showing which parts of the app is optimal and not.
