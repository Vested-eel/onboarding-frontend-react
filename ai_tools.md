Which AI tools did you try?

  - I have tried all AI coding Assistant.

What worked well? What didn’t?

 -  Personally, I think all coding assistant worked out well, it is that, if using simple problem solving, I think they all do well, the difference is the efficiency on how they present it differs. Claude is by far the best, on how concise it is when explaining/debugging, next is chatgpt, it has more or less the same as claude but with a bit more words to say that muddies it, if only a little bit. and lastly github copilot, It does the tasks as well as the others but it does add fluff to its explanation sometimes overwhelming the user.

When do you think AI is most useful for coding?

  - Only for simplifying repetitive tasks, debugging and learning, it acts as a hand holder when trying to learn something new, and idea exploration, as AI can sometimes shed some light on a path not aware to the developer.
